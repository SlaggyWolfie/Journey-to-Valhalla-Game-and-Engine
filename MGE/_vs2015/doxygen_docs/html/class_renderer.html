Renders a specific GameObject in the world using a specific camera. Convenience method for calling render (world, gameobject, material, modelmatrix, viewmatrix, projectionmatrix, recursive)

Parameters
pWorldthe world the mesh is a part of, can be used to retrieve lighting information
pGameObjectthe gameobject to render, will use the this gameobject's world transform for the model matrix
pMaterialthe material to use, is passed on to render (world, gameobject, material, modelmatrix, viewmatrix, projectionmatrix, recursive)
pCameraused for the view and perspective matrix
pRecursiverender everything recursively
